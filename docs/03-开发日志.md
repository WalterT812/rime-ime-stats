# 开发日志

## 2026-06-12 项目启动（v0.1）

**需求**（Walter）：自己的中英文输入法；统计按键次数、中文/英文字数；游戏内不受影响；体验对标微软拼音。

**决策**：
- 技术路线选定"Rime 引擎定制"（用户确认），放弃从零写 TSF——理由见 01-方案设计.md
- 统计展示选定"托盘图标 + 弹出面板"
- 中文方案：雾凇拼音（rime-ice）为主，朙月拼音简化字兜底

**完成**：
- rime-config：default/weasel custom 配置、游戏 app_options（14 个常见游戏进程）、commit_logger.lua 上屏日志
- stats-app：ime_stats.py（pynput 钩子 + 日志读取 + SQLite + pystray/tkinter 面板）
- scripts：rime-ice 一键安装、配置一键部署（带备份）
- docs：方案设计、安装指南

**自查修复**（提交前代码审查发现）：
- 钩子回调内原本直接调 Win32 查询（最多阻塞 50ms）——低级钩子回调阻塞会拖慢全系统按键，违背"游戏无影响"目标。已改为回调只入队，统计逻辑移到工作线程，并对进程名/IME 模式查询做 1s/0.2s 缓存
- `SendMessageTimeoutW` 的 lpdwResult 原用 32 位 DWORD，64 位系统下会越界写，改为 `c_size_t`
- 托盘图标 PIL 默认字体不支持中文"字"，改为加载 msyh.ttc 并留降级

**待办 / 已知问题**：
- [ ] IME 中英模式检测在纯 TSF 场景可能失败（英文数轻微高估），观察实际效果
- [ ] 后续可选：统计面板加图表、C# 重写统计程序降内存
