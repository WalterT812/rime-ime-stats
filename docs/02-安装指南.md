# 安装指南

## 第 1 步：安装小狼毫（Weasel）

1. 打开 https://rime.im ，下载 Windows 版（小狼毫）安装包并安装
2. 安装完成后注销或重启一次，确保输入法注册成功
3. 按 `Win + 空格` 应能切换到"中州韵"输入法

## 第 2 步：安装雾凇拼音词库

在本项目目录打开 PowerShell：

```powershell
cd scripts
.\install_rime_ice.ps1
```

如提示脚本被禁止运行，先执行：`Set-ExecutionPolicy -Scope Process Bypass`

## 第 3 步：部署本项目配置

```powershell
.\deploy_rime_config.ps1
```

然后**右键任务栏的小狼毫图标 →「重新部署」**。等待几秒（首次部署要编译词库，可能 1-2 分钟），部署成功会有通知。

验证：随便找个地方打 `nihao`，应出现"你好"；打字后检查 `%APPDATA%\Rime\commit_log.txt` 是否有记录——有就说明统计日志正常。

## 第 4 步：启动统计程序（推荐 C# 版）

### A. C# 版（主力，常驻内存更低，推荐）

装好 .NET 8 SDK（https://dotnet.microsoft.com/download）后，**推荐一键安装**：

```powershell
cd ..\scripts
.\install.ps1     # 编译 + 安装到 %LOCALAPPDATA%\IMEStats + 注册自启/看门狗 + 启动
```

这样程序装在稳定位置、与源码仓库解耦——**今后移动/重命名/删除本仓库都不影响运行**
（数据本就在 `%APPDATA%\IMEStats`）。任务栏出现蓝色"字"图标即成功，启动会自动弹面板。

> 只想编译、不安装：`.\build_csharp.bat` → `stats-app\IMEStatsSharp.exe`，再 `.\enable_autostart.bat`。

词频/喂词仍由 Python 的 `word_worker.py` 负责（C# 版每 6 小时及托盘菜单会自动拉起它），
若要喂词功能，仍需 `pip install -r requirements.txt` 装好 jieba/pypinyin。

### B. Python 版（参考实现，与 C# 共用数据，随时可互换）

```powershell
cd ..\stats-app
pip install -r requirements.txt
pythonw ime_stats.py
```

> 两版共用 `%APPDATA%\IMEStats\stats.db` 与单实例锁，**不能同时开**——
> 换版本前先右键另一个托盘图标退出。

**开机自启 + 看门狗**：双击 `scripts\enable_autostart.bat`，它会自动按
**C# 版 > Python 打包 exe > Python 脚本**的优先级注册，无需手动选。

> C# 版还可直接在**托盘右键菜单**里勾选「开机自启动」开/关（写启动文件夹），
> 比跑 bat 更方便；菜单里另有「导出 CSV」「打开数据文件夹」。

## 日常使用

| 操作 | 按键 |
|------|------|
| 切换中/英 | 左 Shift |
| 翻页 | `-` / `=` 或 PgUp/PgDn |
| 方案选单 | Ctrl + ` |
| 查看统计 | 点托盘"字"图标 |
| 游戏前（可选） | 托盘右键 → 暂停统计 |

## 添加你自己的游戏

编辑 `rime-config/weasel.custom.yaml` 的 `app_options`，加一行（进程名小写，任务管理器 → 详细信息可查）：

```yaml
    你的游戏.exe: { ascii_mode: true }
```

再运行一次 `deploy_rime_config.ps1` 并重新部署。

## 常见问题

- **打不出中文**：先「重新部署」；仍不行看 `%APPDATA%\Rime\` 下有无 `rime_ice.schema.yaml`（没有则重跑第 2 步）
- **游戏里弹候选框**：把游戏 exe 加入 app_options（见上）
- **统计面板中文字数不增长**：确认正在用小狼毫打字（统计只覆盖本输入法上屏的中文）
- **担心反作弊**：游戏前托盘暂停统计即可；输入法本身是系统正规 TSF 输入法，与微软拼音同级别

## 进阶

- **C# 版（v1.2，主力）**：见上方第 4 步 A。数据库/日志/互斥锁与 Python 版完全共用，
  钩子与统计口径已与 Python 对齐（快捷键不计英文、按字节读日志不漏字、CJK 含扩展区）
- **Python 打包 exe**：双击 `scripts\build_exe.bat` → `stats-app\IMEStats.exe`（约 50MB，含分词库），
  无 Python 环境也能跑；`enable_autostart.bat` 在没有 C# exe 时会回退到它
- **每周词库更新**：双击 `scripts\enable_weekly_dict_update.bat`，每周日 12:00 自动拉雾凇最新词库并重新部署
- **数据位置**：统计数据库统一在 `%APPDATA%\IMEStats\stats.db`（WAL），上屏日志在
  `%APPDATA%\Rime\commit_log.txt`，全部本地，不上传
